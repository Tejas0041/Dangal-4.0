## Quick Start

Navigate to the client folder and follow the instructions:

```bash
cd client
npm install
npm run dev
```

Visit `http://localhost:5173` to view the app.

## Build for Production

```bash
cd client
npm run build
```

The production-ready files will be in `client/dist/`.
