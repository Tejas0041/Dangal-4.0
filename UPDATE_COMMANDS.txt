# Commands to Update Your VPS Deployment

## Step 1: Pull Latest Code
cd /var/www/Dangal-4.0
git pull origin main

## Step 2: Update Backend
cd /var/www/Dangal-4.0/server
npm install
pm2 restart dangal-backend

## Step 3: Update Client
cd /var/www/Dangal-4.0/client
npm install
npm run build

## Step 4: Update Admin
cd /var/www/Dangal-4.0/admin
npm install
npm run build

## Step 5: Verify Everything is Running
pm2 status
pm2 logs dangal-backend --lines 20

## Step 6: Test in Browser
# Open in incognito mode:
# https://dangal2k26.online
# Try signing in with Google

## If Issues, Check Logs:
pm2 logs dangal-backend
tail -f /var/log/nginx/error.log

## Quick Restart Commands (if needed):
pm2 restart dangal-backend
systemctl reload nginx
